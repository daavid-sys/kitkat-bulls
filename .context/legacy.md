# Reonic Legacy Flow (What Exists Today)

Source: reonic_legacy_demo.json (transcription of official Reonic product demo)

## Current process: 17 steps, ~30 min, installer-operated

1. Lead funnel: enter address, select house (30s)
2. Energy profile: household size, consumption, EV, heating (1 min)
3. Product selection: PV, battery, charger checkboxes (30s)
4. Inquiry submission: contact info, ownership, consent (30s)
5. Portal: installer opens lead in CRM (2 min)
6. Pre-qualification call: customizable script, manual notes (5-10 min)
7. Create offer (30s)
8. 3D editor: load map, switch Google/Apple imagery (30s)
9. **MANUAL ROOF DRAWING**: trace edges with mouse, adjust shape, use ruler (3-5 min)
10. **MANUAL OBSTRUCTION MARKING**: add chimney, vents (1 min)
11. **MANUAL MODULE PLACEMENT**: drag panels onto roof (2 min)
12. Inverter selection: match to system size, draw string, assign MPP tracker (2 min)
13. Component planning: import package or add individually (2-3 min)
14. Battery storage: add via planning package (1 min)
15. Review simulations: energy flows, break-even, economy (1 min)
16. PDF offer generation: customize layout, preview (2 min)
17. Send & sign: digital or physical signature (1-2 min)

## What we're killing with AI
- Steps 9-11 (manual roof + panels): replaced by Solar API auto-detection + placement algorithm
- Step 6 (pre-qual call): homeowner self-qualifies in onboarding flow
- Steps 12-13 (component selection): AI selects from Reonic's database
- Step 16 (PDF generation): replaced by live interactive proposal

## What stays (recontextualized)
- Address input (entry point)
- Energy profile (drives system sizing, but with smart defaults)
- Product intent (what does the homeowner want)
- Financial analysis (core value prop — instant, not manual)
- Legal consent (regulatory)
- Offer document (live page, not static PDF)

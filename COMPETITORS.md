# Competitor Landscape

The space is crowded in the US, but the existing tools are all **installer-facing**, not homeowner self-serve.

| Tool | What it does | 3D approach | Self-serve? |
|------|-------------|-------------|-------------|
| **Aurora Solar** | #1 solar design software. Address + electric bill → 3D design in 15 sec | LiDAR + HD imagery + EagleView aerial | No — installer tool. Homeowner sees a PDF proposal |
| **OpenSolar** | Free end-to-end design + proposal | Similar to Aurora, EagleView integration | No — installer-facing |
| **SurgePV** | Fastest auto-roof detection (15-30 sec) | AI + LiDAR, ±1° pitch accuracy | No |
| **Arka 360** | Sends virtual survey link to homeowner | Homeowner uploads photos → AI 3D model | **Closest to self-serve** |
| **Solargraf** | Full design + proposal in 3 min | Satellite imagery | No |
| **Google Project Sunroof** | Sunlight/savings estimate for an address | 2D heatmap, no panel placement | Self-serve but **no 3D**, no proposal |

## The Gap We're Filling

Nobody combines **homeowner self-serve** + **real 3D visualization of their house** + **complete proposal with costs/ROI**.

- Aurora is closest but it's an installer tool.
- Google Sunroof is self-serve but flat 2D with no proposal.
- We're building the thing that bridges both.
